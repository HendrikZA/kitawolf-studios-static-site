Help Center

Frequently Asked Questions

Will crawling slow down my website performance?

No. The crawler engine utilizes throttling parameters and chunk-based scheduling. It makes internal post loops and HTTP requests asynchronously using local cron execution, checking links in small sets to avoid page latency for regular users or exceeding hosting resource limits.

Can I edit links directly in Gutenberg/Page Builders?

Yes. The Inline Link Editor updates links directly inside your database records. It is fully compatible with the WordPress Block Editor (Gutenberg) as well as classic page builders (Divi, Elementor, WPBakery) because it targets the saved text contents safely.

Is there a limit on how many links it checks?

There is no arbitrary limit built into the plugin. The checker can audit thousands of links across massive databases, limited only by your hosting server's capacity to process cron routines.
